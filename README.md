# mer4t - System Gauge

Windows için gerçek zamanlı sistem izleme uygulaması.

## Özellikler

* CPU ve GPU kullanım takibi
* CPU ve GPU sıcaklık takibi
* Saat hızı ve fan devri takibi
* Anakart, RAM ve SSD bilgileri
* İndirme ve yükleme hızı takibi
* Windows ile otomatik başlatma
* Sistem tepsisine küçültme
* Pencere konumunu hatırlama

## İndirme

[En son sürümü indirmek için Releases sayfasına gidin.](https://github.com/mer4t/mer4t-System-Gauge/releases/latest)

## Sistem Gereksinimleri

* Windows 10 veya Windows 11
* 64-bit işletim sistemi
* Uygulama yönetici yetkisiyle çalışır

## Kullanım

Uygulamayı çalıştırmadan önce **donanım sensörlerini kullanan diğer uygulamaları** kapatın.
